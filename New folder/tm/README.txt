Instructions for executing the tm application:

1. Make sure you have Python installed on your system (version X.X.X or later).
2. Open a terminal or command prompt.
3. Navigate to the "tm" subdirectory.
4. Run the following command to execute the application:

   python TestManager.py

5. Go to localhost:3002 on your browser.
6. Use the credential stored in user.txt (Username: user / Password: pass).
